# EVA — Дорожная карта

## Фаза 0: Стабильный язык (май 2026)

**Цель:** модель порождает связный русский текст.

| Задача | Где | Время | Статус |
|--------|-----|-------|--------|
| Токенизатор 50K слов | ПК | 20 мин | ✅ Готово |
| Обучение 50K шагов Wikipedia | ПК | 80 ч | 🔄 Шаг 1000/50000 |
| Gen-test каждые 1000 шагов | ПК | — | ✅ Готово |
| Kaggle/Colab GPU-тренировка | Облако | 2–4 сессии | ⬜ Ждёт верификации |
| Инструктивное дообучение | ПК | 2 ч | ⬜ После языка |
| **Результат:** `fcf.bat --interactive` → осмысленные ответы | | | |

## Фаза 1: Активное поглощение (июнь 2026)

**Цель:** модель сама идёт за знаниями.

| Задача | Компоненты | Строк кода |
|--------|-----------|------------|
| Авто-поиск в Wikipedia при низком SRG | CuriosityLoop → DataManager Wikipedia | ~30 |
| Авто-извлечение фактов из статьи | ConceptMiner базовая версия | ~50 |
| Авто-дообучение LoRA домена | AutoTrainer + DomainRegistry | уже есть |
| Замыкание когнитивной петли: сканирование → поиск → обучение → проверка | EVACognitiveLoop | ~200 |
| **Результат:** спросил про квантовую физику → не знает → идёт в Wiki → учит → отвечает | | |

## Фаза 2: Концептуальное мышление (июль 2026)

**Цель:** модель обнаруживает пробелы в знаниях и формулирует гипотезы.

| Задача | Компоненты |
|--------|-----------|
| Сканирование кластеров FAISS на лакуны | ConceptMiner + HDBSCAN |
| Генерация гипотез через State Algebra | z(известное) → z(неизвестное) |
| Валидация гипотез через Wikipedia | ContradictionDetector |
| Визуализация графа знаний | `--graph` команда |
| **Результат:** модель сама предлагает: «Я обнаружил, что нет связи между X и Y. Проверить?» | |

## Фаза 3: Автономный рост (август 2026)

**Цель:** модель сама решает когда расти вширь и вглубь.

| Задача | Компоненты |
|--------|-----------|
| GrowthController в production | пороги по реальным метрикам |
| Авто-кристаллизация слоёв | LayerCrystallizer по failed_queries |
| Sleep Mode в production | авто-дистилляция, авто-слияние |
| Multi-layer inference | sequential pipeline 2–4 слоя |
| **Результат:** модель растёт и сжимается без участия человека | |

## Фаза 4: Создание нового (сентябрь 2026+)

**Цель:** модель предугадывает существование неизвестных концептов.

| Задача | Компоненты |
|--------|-----------|

| **Результат:** «Предлагаю новый концепт X на основе синтеза A и B. Проверить?» | |
